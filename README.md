# Fernandez_etal_2026_ValuingKelpForest_forSharks
Here you will find the data of the manuscript: Valuing kelp as nursery grounds for sharks: "Influence of kelp attributes and forest configuration on the oviposition site of a shark species in Chile" published in Diversity journal 2026.

The file "egg_morpho.csv" is data about egg-bering kelp individuals at each site: total number of eggs, separated by each egg-clutch found within the kelp; depth of the kelp, and Holfast Diameter (DD), number of stipes (NE), and total lenght (LT) in cm.

The file "kelp_density.csv" has the abundance of juvenile and adult kelp along transect at each site

The file "kelp_morpho.csv" has the data of the morphology of non-egg bearing kelp (same as "egg_morpho.csv") for each kelp within a quadrant inside a transect at each site.
